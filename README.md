# fingermé

A golang based FINGERD(8) server for serving my my résumé as a .plan file.

## Usage

Query the server via a FINGER(1) client:

    $ task serve
    $ finger retr0h@localhost

Install dependencies

    $ task deps

Test packages:

    $ task unit

Test all:

    $ task test

List targets:

    $ task
